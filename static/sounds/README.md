# ゲームサウンドファイル

このフォルダに以下のサウンドファイルを追加してください：

| ファイル名 | 用途 | 推奨 |
|-----------|------|------|
| `click.mp3` | ボタン/タップ時 | 短く軽快な「ポン」「ティック」音 (50-100ms) |
| `buy.mp3` | 施設購入時 | コイン/チャリン音 (100-200ms) |
| `levelup.mp3` | アップグレード購入時 | レベルアップ/達成音 (300-500ms) |
| `milestone.mp3` | マイルストーン達成時 | ファンファーレ/特別な達成音 (500-1000ms) |
| `prestige.mp3` | 転生時 | 荘厳/壮大な転生音 (1000-2000ms) |

## 無料サウンド素材サイト

- **効果音ラボ** - https://soundeffect-lab.info/
- **ポケットサウンド** - https://pocket-se.info/
- **Freesound** - https://freesound.org/
- **Mixkit** - https://mixkit.co/free-sound-effects/

## 推奨フォーマット

- **形式**: MP3
- **サンプルレート**: 44.1kHz
- **ビットレート**: 128kbps以上
- **ファイルサイズ**: 各50KB以下推奨
