# 実装済みコマンド一覧

## 🛠 管理者(親用)
| コマンド | 説明 |
| --- | --- |
| **タスク追加 [タイトル] [報酬]** | 新しいお手伝いタスクを追加します。<br>例: `タスク追加 風呂掃除 300` |
| **商品追加** | ショップに商品を追加するためのGoogleフォームのURLを表示します。 |
| **勲章授与** / **バッジ** | ユーザーに特別なバッジ（勲章）を授与します。<br>ユーザー選択 → バッジ選択 の順で操作します。 |
| **ポイント付与** | ユーザーにポイントを付与します。<br>ユーザー選択 → ポイント入力 → 理由入力 の順で操作します。 |
| **ポイント修正** | ユーザーのポイント残高を直接修正します。<br>ユーザー選択 → 新しい残高を入力 の順で操作します。 |
| **状況** / **ステータス** | ユーザーのステータス（学習状況、ランク、ポイント）を確認します。<br>「誰のステータスを確認しますか？」というメニューが表示されます。<br>「全体の履歴」を選ぶと、直近の取引履歴が表示されます。 |
| **コマンド** | 管理者用コマンド一覧を表示します。 |

## 👦 一般(子供用)
| コマンド | 説明 |
| --- | --- |
| **勉強開始** | 勉強セッションを開始します。教科選択メニューが表示されます。 |
| **勉強終了** | 勉強セッションを終了し、時間と集中度を記録します。 |
| **ショップ** / **使う** / **ポイント交換** | ポイント交換ショップを表示します。 |
| **ジョブ** / **お手伝い** | 現在募集中のジョブ（お手伝い）リストを表示します。 |
| **状況** / **ステータス** | 自分のステータス（ランク、ポイント、学習履歴）を表示します。<br>「詳細ステータス」ボタンで週間・月間の学習グラフが見られます。 |
| **ガチャ** | ガチャメニューを表示します（現在は未実装メッセージが表示されます）。 |
| **ヘルプ** / **使い方** | 子供用の使い方ガイドを表示します。 |

## 💻 開発者用
| コマンド | 説明 |
| --- | --- |
| **!reset** / **!init** | 自分自身のユーザーデータをリセットします（開発用）。 |
